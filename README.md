# OpenCV-Color-Filtering
# Real-Time BGR Color Filtering
Bu proje, OpenCV kütüphanesi kullanarak videolardaki belirli renkleri (Mavi, Yeşil, Kırmızı) 
dinamik olarak filtrelemeyi sağlar.

## Kullanılan Teknolojiler
* Python
* OpenCV
* NumPy

## Nasıl Çalıştırılır?
1. OpenCV kütüphanesini indirin: `pip install opencv-python`
2. Kodu çalıştırın: `Color_Filtering.py`
3. Trackbar üzerinden renk aralıklarını ayarlayarak nesneleri filtreleyin.
